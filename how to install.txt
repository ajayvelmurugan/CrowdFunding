Requirements:
  1. Install Node Js 
  2. Install Metamask Plugin in chrome or firefox browser
 
Steps to Install:
  1. Extract the CrowdFunding.zip file
  2. Open Terminal in CrowdFunding Folder
  3. Type the following Command
          npm install
  4. Compile the contract using the following command
         node /ethereum/compile.js  
  5. Deploy the contract using the following command
         node /etherem/deploy.js
  6. Now copy the deployed address to factory.js file
  7. To run the contract type the following command
         npm run dev
  8. Open the browser and go to the following address
        localhost:3000
